# 易读 AI 决策中心设计

## 目标

将实时页从原始 JSON 调试器改为玩家可快速理解的战斗摘要，同时保留 Debug 页用于排查完整数据。

## 实时页

- 游戏状态：显示角色生命、格挡、能量、金币、楼层和回合；敌人显示生命、格挡、意图和预计伤害；手牌显示名称、费用、类型以及是否可用。
- 当前决策：用自然语言突出最终动作，显示简短理由、模型耗时和排名前三的候选方案。
- 实时轨迹：将遥测事件翻译为中文短句，只显示时间、状态、动作和失败原因，不显示完整 payload。
- 空闲、主菜单、断线和状态过期均提供明确空状态。

## 历史与 Debug

- 历史页显示对局摘要列表，并允许展开对局详情。
- Debug 页保留完整 snapshot JSON，原始信息不丢失。

## 约束

- 不修改 `18888` Mod 协议和 `18889` Dashboard API。
- 使用原生 HTML、CSS 和 JavaScript，不增加前端依赖。
- 所有动态文本通过 `textContent` 写入，避免注入风险。
- 桌面三栏，窄屏单栏；文本不能溢出或重叠。

## 验证

- 自动测试检查语义化容器、Debug 容器和前端安全写入方式。
- 完整 Python 测试通过。
- 用 Headless Chrome 截图检查桌面布局、中文显示、空状态和溢出。
